Help Desk My project

Welcome to the Help Desk system develop by me while studying Full Stack Development.
This is a call management system simulating a simple Help Desk developed with Angular 12 and Spring Boot 2.
The system is running on cloud: front in Vercel and back in Heroku.
The project code can be accessed by clicking in the link GitHub in the side bar.
Please feel confortable to test it and to leave any suggestions.
